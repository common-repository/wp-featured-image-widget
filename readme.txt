== Description ==

Provides a widget that displays the featured image for a post or page with custom settings.

The widget features.

1. Show Title.
2. Set Image size to display.
3. Enable/Disable post/page link.
4. Enable/Disable parent featured image. 
5. Enable/Disable slider 
6. Enable/Disable slider autoplay.
7. Enable/Disable autoplay infinite.
8. Enable/Disable slider arrows 
9. Set Image slider animation. 
== Installation ==

It's dead simple:

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Browse to Appearance->Widgets, add the widget, and configure the options

== Frequently Asked Questions ==

= How many featured images can we show? =

You may use as many featured images you want.
